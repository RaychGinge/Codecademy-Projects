HTML Cheatsheet

Tag What do I do?
<!DOCTYPE html> Must always be the first line of code
<html> Contains all of the html code
<head> Information about the webpage goes in this tag
<title> Adds a title to the browser tab for the webpage
<a> Used to link internal/external pages, or content on the same page
<body> Any visibile content goes within this tag
<h1>
<h2>
<h3>
<h4>
<h5>
<h6>		 Headings and sub-headings
<p>
<span>
<div>		 Specify text or blocks
